# Glossary

| Term | Definition | Scope |
|---|---|---|
| Active node | The system graph node currently being worked on. | project governance |
| Claim | A statement evaluated by evidence, derivation, simulation, test, or observation. | evidence |
| Contract | A boundary preserving meaning across modules, teams, disciplines, or abstractions. | design |
| Evidence | A durable record testing or informing a claim. | evidence |
| Model gate | The scripted policy check before durable AI-assisted writes. | model governance |
