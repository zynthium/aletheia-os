# AletheiaOS Bootstrap

This repository has been initialized with AletheiaOS.

Before implementation work, define the project charter, root system graph thesis, active state, and first decision record. Do not treat existing files or chat history as automatically true project state.

Minimum bootstrap steps:

1. Read `.aletheia/START_HERE.md`.
2. Fill `.aletheia/governance/CHARTER.md`.
3. Update `.aletheia/state/SYSTEM_GRAPH.yaml`.
4. Update `.aletheia/state/SKELETON.yaml`.
5. Update `.aletheia/state/ACTIVE_STATE.md`.
6. Create a decision record under `.aletheia/decisions/`.
7. Run `python3 .aletheia/bin/validate.py`.
8. Finalize with `python3 .aletheia/bin/bootstrap_finalize.py`.
