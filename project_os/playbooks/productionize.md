# Playbook — Productionize

Use when moving from idea/prototype to operational system.

Required checks:

- linked active node;
- interface contract;
- evidence record sufficient for stage transition;
- monitoring or validation plan;
- rollback/fallback logic;
- data/provenance traceability;
- failure mode review;
- decision record if crossing a major boundary.

Do not productionize if the implementation changes the meaning of the original claim without recording it.
