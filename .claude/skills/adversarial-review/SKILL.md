# Adversarial Review Skill

Look for:
- hidden assumptions;
- invalid proxy metrics;
- overfitting or result chasing;
- implementation changing theoretical meaning;
- missing controls;
- unmodeled risk;
- interface mismatch;
- parent-assumption invalidation;
- operational infeasibility.

Classify issues by severity and blocker type A/B/C/D. Recommend durable state updates.
