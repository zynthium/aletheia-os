# Productionize Skill

Before productionization, require:
- linked active node;
- evidence threshold met;
- interface contract;
- monitoring/validation plan;
- failure modes;
- rollback or fallback;
- reproducibility path;
- decision record if boundaries or objectives change.

Output affected files, tests/checks, contracts, risks, and checkpoint status.
