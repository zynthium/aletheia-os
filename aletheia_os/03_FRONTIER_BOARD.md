# Frontier Board

Use this as a research/design/engineering portfolio board. It prevents the newest or loudest branch from automatically receiving the most attention.

## Scoring fields

| Field | Meaning |
|---|---|
| strategic_value | If solved, how much does it advance the mission? |
| dependency_unblock_value | Does it unlock many other branches? |
| uncertainty_reduction_value | Does it reduce critical uncertainty? |
| feasibility | Can progress be made now? |
| evidence_quality | Are current facts reliable? |
| implementation_cost | Time/capital/compute/complexity cost. |
| drift_risk | Risk of pulling attention away from the root objective. |
