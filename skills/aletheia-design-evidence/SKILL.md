# Aletheia Design Evidence

For each important claim, record:

- claim;
- method;
- evidence type;
- expected observation;
- invalidation criteria;
- interpretation rule;
- graph impact;
- follow-up decision.

Do not treat implementation success, benchmark output, or model confidence as self-explanatory evidence.
